# 旁白风格参考

本文档提供旁白创作的风格参考，用于生成分镜故事板时参考。

## 短剧旁白风格

### 风格特点

1. **简洁有力**：每句不超过20字
2. **情感饱满**：传达角色内心或场景氛围
3. **推动节奏**：配合画面快速推进剧情
4. **留白悬念**：适当停顿，制造张力

### 情感基调分类

#### 紧张悬疑
```
黑暗中，脚步声越来越近。
他知道，再也回不去了。
真相，就藏在那扇门后。
```

#### 热血激昂
```
这一刻，他终于明白了战斗的意义。
燃烧吧，不灭的意志！
胜负，就在此刻！
```

#### 温馨治愈
```
阳光洒在她的笑容上。
平凡的日子，也有不平凡的温暖。
这就是，家的味道。
```

#### 悲伤感人
```
她转身的那一刻，泪水滑落。
有些话，终究没能说出口。
再见，是最遥远的距离。
```

#### 搞笑幽默
```
他万万没想到，事情会变成这样。
这下，可闹大了。
emmm…这剧本不对啊！
```

## 分镜旁白创作原则

### 1. 与画面配合

**❌ 不好的示例**（重复画面信息）：
```
画面：角色打开门
旁白：他打开了门
```

**✅ 好的示例**（补充画面外信息）：
```
画面：角色打开门
旁白：门后的真相，即将揭晓
```

### 2. 节奏控制

**快节奏场景**（动作、追逐、打斗）：
- 短句
- 断句
- 强调词
```
跑！
不能停！
身后，是绝望的深渊。
```

**慢节奏场景**（情感、回忆、对话）：
- 长句
- 流畅
- 抒情
```
记忆中的那个午后，阳光透过树叶，洒在她微笑的脸上，温柔得像一首诗。
```

### 3. 反转铺垫

**第一次反转前**：
```
一切都按计划进行。
胜利，近在眼前。
```

**反转时刻**：
```
直到，那个人出现。
```

**第二次反转前**：
```
这一次，他不会再输了。
```

**反转时刻**：
```
但他忘了，对手也在成长。
```

### 4. 开头钩子

前3秒必须抓住观众，旁白示例：

**悬疑类**：
```
如果我告诉你，你身边的人都是假的呢？
```

**热血类**：
```
今天，我要改变一切。
```

**爱情类**：
```
那一刻，我明白了什么叫心动。
```

**惊悚类**：
```
尖叫声，在午夜响起。
```

## 分镜旁白时长控制

| 分镜时长 | 旁白字数建议 | 示例 |
|---------|-------------|------|
| 1-2秒 | 5-8字 | "他来了。" |
| 2-3秒 | 8-12字 | "黑暗中，脚步声越来越近。" |
| 3-5秒 | 12-20字 | "她转身的那一刻，泪水终于忍不住滑落。" |
| 5秒以上 | 20-30字 | "记忆中的那个午后，阳光透过树叶，洒在她微笑的脸上，温柔得像一首诗。" |

## 旁白与对话的配合

### 规则
1. **避免同时出现**：旁白和对话不在同一时刻
2. **旁白承上启下**：连接不同对话场景
3. **对话推动剧情**：角色互动
4. **旁白渲染氛围**：情感深化

### 示例结构

```
分镜1（2秒）：
  画面：角色A独自站在窗前
  旁白：雨停了，但他的心还在下雨。

分镜2（3秒）：
  画面：角色B推门进入
  对话：角色B："你还好吗？"

分镜3（2秒）：
  画面：角色A转身
  对话：角色A："我很好。"

分镜4（2秒）：
  画面：特写角色A的眼神
  旁白：谎言，从来不需要太多修饰。
```

## 爽点/反转时刻的旁白

### 爽点高潮
```
来吧，让你见识真正的力量！
这一刻，所有人都看呆了。
翻盘，从现在开始！
```

### 反转揭秘
```
原来，一切都是他的局。
真相，远比想象的残酷。
她笑了，那是胜利者的笑容。
```

### 情感爆发
```
他终于说出了那句话。
泪水，是最诚实的语言。
这一刻，值得用一生去等待。
```

## 不同类型短剧的旁白风格

### 古风/仙侠
```
剑起，风云动。
天地之间，唯吾独尊。
红尘万丈，不过一场梦。
```

### 现代都市
```
这座城市，从不缺少故事。
爱情和面包，她选择了……
梦想，是唯一的奢侈品。
```

### 科幻未来
```
人类的末日，就在今天。
系统重启，倒计时：10…9…8…
新世界，即将降临。
```

### 青春校园
```
那年夏天，我们都还年轻。
青春，就是用来挥霍的。
毕业那天，我终于告诉了她。
```

## 旁白禁忌

❌ **避免以下情况**：

1. **过度解释**：不要说出显而易见的画面内容
2. **信息过载**：一句话包含太多信息点
3. **情感不符**：旁白情感与画面氛围矛盾
4. **时长失配**：旁白太长，分镜时长不够
5. **重复对话**：旁白重复角色对话内容

## 创作流程建议

1. **先写画面**：明确每格分镜的视觉内容
2. **再写对话**：角色互动
3. **最后加旁白**：补充情感和氛围，避免重复
4. **反复朗读**：检查节奏和流畅度
5. **配合时长**：确保旁白能在分镜时长内说完

---

**提示**：旁白是分镜的灵魂，不是解说员。要传达情感、制造悬念、推动节奏，而不是描述画面。
